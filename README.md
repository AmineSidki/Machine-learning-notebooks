# Machine Learning Notebooks

Ce dépôt contient une série de notebooks Jupyter explorant divers domaines du Machine Learning, allant de la vision par ordinateur au traitement du langage naturel (NLP).

## Structure du Projet

Le dépôt est organisé par thématiques, correspondant aux différents travaux pratiques effectués :

- **fashion_cnn.ipynb**: Implémentation d'un réseau de neurones convolutif (CNN) pour la classification d'images sur le jeu de données Fashion MNIST.
- **nlp_transformers.ipynb** : Exploration du traitement du langage naturel via l'architecture Transformers (fine-tuning, classification).
- **timeseries_lstm.ipynb** : Modélisation et prédiction de séries temporelles utilisant les réseaux LSTM.
- **tinyml_fuel.ipynb** : Projet axé sur le TinyML pour l'analyse et la prédiction de la consommation de carburant.

## Prérequis

Pour exécuter ces notebooks, assurez-vous d'avoir un environnement Python configuré avec les bibliothèques principales suivantes :

- PyTorch
- Hugging Face Transformers
- Scikit-learn
- Pandas
- Matplotlib / Seaborn

Dépôt créé dans le cadre des projets d'apprentissage automatique.
